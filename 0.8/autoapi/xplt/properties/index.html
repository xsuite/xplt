Class for property which can directly be accessed from data

values(data, mask=None, *, unit=None)[source]#

Get masked data for this property

Parameters

  • data (any) – The data object providing the values

  • mask (None | Any | callable) –

    An optional mask to apply to the values. Can be None, a slice, a binary mask or a callback. If a callback, it must have the signature (mask_1, get) -> mask_2 where mask_1 is the binary mask to be modified, mask_2 is the modified mask, and get is a method allowing the callback to retrieve particle properties in their respective data units. .. rubric:: Example

    def mask_callback(mask, get):

    mask &= get(“t”) < 1e-3 # all particles with time < 1 ms return mask

  • unit (str) – The unit to convert the data to, must be compatible with this property
